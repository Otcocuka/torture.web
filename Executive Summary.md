Executive Summary: Intelligent Learning & Productivity System
Project: AI-powered browser-based SPA integrating personal productivity (GTD, Pomodoro, Kanban, habit tracking) with a cognitive learning engine grounded in spaced repetition and active recall.

Key Technical Contributions:

LLM-driven atomic knowledge extraction from educational texts with automatic topic clustering
Step-based spaced repetition algorithm with adaptive intervals [1, 3, 7, 14, 28] days
Adaptive quiz engine: LLM-scored open-ended answers (0–100% scale), retry-on-failure logic
Knowledge Avatar: visual knowledge graph with per-concept mastery tracking and topic-based navigation
Structured research data pipeline with JSON export for longitudinal analysis
Technology Stack: Vanilla JavaScript, HTML5, CSS3, Node.js/Express, DeepSeek LLM API. Zero build-step dependencies, 100% client-side state management via LocalStorage. Styling via Tailwind CSS (CDN).

Research Foundation: Bachelor's thesis at Russian State Social University, 2025. Simulation modeling on open educational data (Student Performance dataset, Kaggle, n=649) suggests 15–25% improvement in retention metrics.

Next Steps (MEXT/G30 Master's): Cross-cultural validation study (Japan + Russia, n=120), biometric integration (EEG, eye-tracking), machine learning for adaptive personalization. Target publications: IEICE, J. Educational Technology & Society.
